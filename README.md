# PYTHON OS

Интерактивная песочница консоли Armbian на Python с виртуальной файловой системой, встроенным nano-редактором, поддержкой 50+ команд и **реальным OTA-обновлением**.

##  Быстрый старт
# если у вас не windows 
```bash
git clone https://github.com/confing-sys/py_OS
cd armbian-webos
python armbian_webos.py
```
